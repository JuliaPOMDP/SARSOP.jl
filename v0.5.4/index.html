SARSOP

A Julia wrapper for the SARSOP partially observabe Markov decision process solver.

Package Features

  • Easy to use interface for the APPL SARSOP package
  • Automtically generated .pomdpx files using the POMDPs.jl interface
  • Automatic policy file parsing and policy generation
